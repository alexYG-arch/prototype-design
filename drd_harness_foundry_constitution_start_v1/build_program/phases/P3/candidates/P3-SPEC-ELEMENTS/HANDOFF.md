## What This Candidate Defines

`P3-SPEC-ELEMENTS` defines how approved source semantics and interaction closure become an atomic, reviewable page element universe. It separates natural language meaning from inventory indexing, requires deductive completion before auxiliary induction, and blocks product-expanding missing surfaces behind Human Gate review.

## Review Focus

1. Confirm that natural language remains the primary semantic authority and inventory rows are only the verification skeleton.
2. Confirm that element rows are atomic and cover more than buttons, states, and input paths.
3. Confirm that deductive completion is explicit and cannot add product capability.
4. Confirm that missing child pages, lower-level surfaces, new roles, permissions, integrations, data scope, or workflows route to Human Gate review.
5. Confirm that downstream patterns, layout, and compiler work cannot invent elements outside this universe or a resolved gap decision.
