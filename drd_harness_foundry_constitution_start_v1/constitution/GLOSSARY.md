# Glossary

- **Harness**：组织多个 Stage、Runtime、Validator 与 Artifact 的受控系统。
- **Foundry**：建设 Harness 的施工体系。
- **Constitution**：不可违反的顶层原则。
- **Contract**：输入、输出、接口和所有权。
- **Rule**：推理与判断规则。
- **Projection**：上游语义如何投射为下游产物。
- **Skill**：Codex 的可复用执行流程，不是事实源。
- **Workpack**：一次 Codex 执行的最小任务单。
- **Capsule**：当前 Workpack 的最小上下文包。
- **SPEC_LOCK**：锁定规范版本及 Hash。
- **BUILD_LOCK**：锁定实现版本、测试和 Hash。
- **Reaction**：可点击元素触发的明确结果。
- **Interaction closure**：所有可点击元素与后续节点均被展开至已有节点或终点。
